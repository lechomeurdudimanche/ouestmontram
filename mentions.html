Sources des données

Cette application utilise les API publiques de Mobilités M (le service d'information des mobilités de l'aire grenobloise).

Les données de transport sont fournies par le SMMAG (Syndicat Mixte des Mobilités de l'Aire Grenobloise) et sont diffusées sous la licence Open Database License (ODbL).

Les données sont à titre indicatif.

Confidentialité & Cookies

Cette application a été conçue dans le respect de votre vie privée :

  • Aucune donnée personnelle n'est collectée, stockée ou revendue.
  • Aucun traceur publicitaire ou outil d'analyse intrusif n'est utilisé.
